1. SCOPE

The goal of this document is to provide the basic tips in order to make the data entry for the Header of the pages.

The Header component is provided by the page template.


2. Header

In order to configure the Footer, follow these steps:

  1. Select the Header Component
  2. Click “Edit” button
  3. In the component dialog, configure these fields:

GENERAL

  • No Cars text: the label that displayed if you clik on “Auto salvate” and there aren’t saved searches;
  • No Searches text: the legal notes title of the section;
  • Prov label: the legal notes description of the section;
  • Km label: the legal notes description of the section;
  • Warranty label: the legal notes description of the section;
  • Saved Cars label: the legal notes description of the section;
  • Saved Searches label: the legal notes description of the section;

SHOWCASES AND DEALERSHIPS

Showcases

  • Showcases label: the label of showcases;
  • Showcases interior title: the interior title of showcases;
  • Showcases interior description: the interior description of showcases;

 

Dealerships

  • Dealerships label: the label of dealerships;

PROFILE

Login and logout

  • Login label: the label of login section;
  • Login text: the text of login section;
  • Login Button label: the login button label of login section;
  • Logout label: the logout label of login section;
  • Login modal title label: the login modal title label of login section;
  • Login modal subtitle label: the login modal subtitle label of login section;

Personal area

  • Hello Message label: the label of hello message;
  • Personal Area label: the label of personal area;
  • Your Profile label: the label of your profile;


MOBILE MAP LINKS


  • Build map list using: choose the page list type:
    • Fixed List: allow the selection of the single pages
    • Child pages: show all the children page of a hub page
  • Option for Fixed List
    • Pages: the path of the page;
