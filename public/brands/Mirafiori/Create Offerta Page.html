1. SCOPE

The goal of this document is to provide the basic tips in order to create the Offerta page in the Mirafiori Outlet portal.

2. HOW TO CREATE THE OFFERTA PAGE

    In order to create the Offerta Page, follow these steps:

  1. Select the language page (e.g. mirafiorioutlet/mirafiorioutlet-it/it)
  2. Click: Create > Create Page
  3. Choose “Mirafiori Outlet Detail Template” as page layout
  4. Click Next
  5. Insert page data (for example, title can be “Offerta”)
  6. Click “Create”;
  7. Click “Done” in the opened popup
