1. Scope

The goal of this document is to provide the full components catalog of the Mirafiori Outlet portal.

Banner

In order to add the “Banner” component into the page follow these steps:

  1. Click toggle side panel on the page ribbon
  2. Select “Components” tab
  3. Choose “Banner” component
  4. Drag and drop the component in the page.
  5. Click on the “Configure” icon in order to set the information of the component.
  6. In the dialog, configure these fields:
General


  • Title: the title of the section;
  • Image: the background image of the section;
  • Brand label: the brand label of the section (visible if image is empty);
  • Description: the description of the section;
  • CTA Label: the CTA Label of the section;
  • CTA Linked page: the linked page the section;

HOME SEARCH

In order to add the Full Editorial component into the page follow these steps:

  1. Click toggle side panel on the page ribbon
  2. Select “Components” tab
  3. Choose “Home search” component
  4. Drag and drop the component in the page.
  5. Click on the “Configure” icon in order to set the information of the component.
  6. In the dialog, configure these fields:
GENERAL

General

  • Headline: the headline text of the section;
  • Showcase Title: the showcase title for the section;
  • Showcase CTA Label: the showcase CTA label for the section;
  • Promotions Title: the title of promotions;
  • Promotions number: the number of promotions displayed;

Promotion 1

  • Showcase Lookup: the showcase lookup page for promotions 1 *;
  • Image: the image for promotions 1 *;

Promotion 2

  • Showcase Lookup: the showcase lookup page for promotions 2 *;
  • Image: the image for promotions 2 *;

* Hight important:

- if you choose “0” in “Promotions number” field, is not required to select a showcase lookup page and a image for Promotions 1 and 2;

- if you choose “1” in “Promotions number” field, is not required to select a showcase lookup page and a image for Promotions 2;

- if you choose “2” in “Promotions number” field, is required to select a showcase lookup page and a image for Promotions 1 and 2;
