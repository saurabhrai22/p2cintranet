1. SCOPE

The goal of this document is to provide the basic tips in order to create the Update password page in the Mirafiori Outlet portal.

2. HOW TO CREATE THE UPDATE PASSWORD PAGE

In order to create the Update password Page, follow these steps:

  1. Select the language page (e.g. mirafiorioutlet/mirafiorioutlet-it/it/)
  2. Click: Create > Create Page
  3. Choose “Mirafiori Outlet Editorial Template” as page layout
  4. Click Next
  5. Insert page data (a title can be “update password”)

  6. Click “Create”;

  7. Click “Done” in the opened popup

3.HOW TO CONFIGURE THE RECOVERY PASSWORD COMPONENT

In order to configure the “Recovery password” component into Update password page, open the page in “Edit Mode” and see related document “AEM – Mirafiori Outlet – Components Catalog”.
