  • The goal of this document is to provide the basic tips in order to create the Map page in the Mirafiori Outlet portal.

    • 2. HOW TO CREATE THE MAP PAGE

      In order to create the Map Page, follow these steps:

    • Select the language page (e.g. mirafiorioutlet/mirafiorioutlet-it/it/)
    • Click: Create > Create Page
    • Choose “Mirafiori Outlet Editorial Template” as page layout
    • Click Next
    • Insert page data (a title can be “Mappa”)

    • Click “Create”;
    • Click “Done” in the opened popup

    3. HOW TO CONFIGURE THE MAP PAGE

    In order to configure the Map page follow these steps:

    • Select for example the “Usato Fiat” page (e.g. mirafiorioutlet/mirafiorioutlet -it/it/map/brands/usato-fiat)
    • Click “Properties”
    • In the “ADVANCED” tab configure these fields:

    • Redirect: the path of showcase page filtered;

    4. HOW TO CONFIGURE THE MAP COMPONENT

    In order to configure the “Map” component into Map page, open the page in “Edit Mode” and see related document “AEM – Mirafiori Outlet – Components Catalog”.
