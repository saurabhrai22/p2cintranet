1. SCOPE

  • The goal of this document is to provide the basic tips in order to create Login page in the Mirafiori Outlet portal.

    • 2. HOW TO CREATE THE LOGIN PAGE

      In order to create the Login Page, follow these steps:

    • Select the language page (e.g. mirafiorioutlet/mirafiorioutlet-it/it)
    • Click: Create > Create Page
    • Choose “Mirafiori Outlet Login Template” as page layout
    • Click Next
    • Insert page data (a title can be “login”)

    • Click “Create”;
    • Click “Done” in the opened popup

    3. HOW TO CONFIGURE THE LOGIN PAGE

    In order to configure the Login page follow these steps:

    • Select the “login” page (e.g. mirafiorioutlet/mirafiorioutlet -it/it/login)
    • Click “Properties”
    • In the “LOGIN” tab configure these fields:

    Banner
    1. Title: the title of banner section;
    2. Subtitle: the subtitle of banner section;
    3. Description: the description of banner section;
    Login Area
    1. Title: the title of login area section;
    2. Email placeholder: the email placeholder label of login area section;
    3. Forgot password: the forgot password label of login area section;
    4. Privacy policy text 1: the first privacy policy text of login area section;
    5. Privacy policy text 2: the second privacy policy text of login area section;
    6. Privacy policy text 3: the third privacy policy text of login area section;
    7. Login CTA label: the login CTA label of login area section;
    Password recovery area
    1. Title: the title of password recovery area section;
    2. Subtitle: the subtitle of password recovery area section;
    3. Description:the description of password recovery area section;
    4. Send CTA label: the send CTA label of password recovery area section;
    5. Success text: the success text of password recovery area section;
    6. Error text: the error text of password recovery area section;
    7. Generic error text: the generic error text of password recovery area section;
    8. Back to login label: the back to login label of password recovery area section;
    Social area
    1. Title:the title of social area section;
    2. Subtitle:the subtitle of social area section;
    Registration banner
    1. Title:the title of registration banner section;
    2. Description:the description of registration banner section;
    3. Register CTA label: the register CTA label of registration banner section;
    4. Image: the image path of registration banner section;



      In order to complete configuration of the Login page follow these steps:

    • Select the “it” page (e.g. mirafiorioutlet/mirafiorioutlet -it/it)
    • Click “Properties”
    • In the “FORM GENERAL” tab configure these fields:

    • privacythirdpartdescription: the privacy third part description;
    • privacytitle: the privacy title;
    • privacyinformationtitle: the privacy information title;
    • closeprivacybutton: the close privacy button label;
