SCOPE

    The goal of this document is to provide the basic tips in order to create the Selected4u page in the Mirafiori Outlet portal.

HOW TO CREATE THE SELECTED4U PAGE

    In order to create the Selected4u Page, follow these steps:

  1. Select the language page (e.g. mirafiorioutlet/mirafiorioutlet-it/it/)
  2. Click: Create > Create Page
  3. Choose “Mirafiori Outlet Editorial Template” as page layout
  4. Click Next
  5. Insert page data (a title can be “Selected4u”)
  6. Click "Create";
  7. Click "Done"; in the opened popup

HOW TO CONFIGURE THE EDITORIAL WARRANTY CERTIFICATE COMPONENT

In order to configure the “Editorial warranty certificate” component into Selected4u page, open the page in “Edit Mode” and see related document “AEM – Mirafiori Outlet – Components Catalog”.
