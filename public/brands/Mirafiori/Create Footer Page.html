1. SCOPE

The goal of this document is to provide the basic tips in order to make the data entry for the Footer of the pages.

The Footer component is provided by the page template.


2. Footer

In order to configure the Footer, follow these steps:

  1. Select the Footer Component
  2. Click “Edit” button
  3. In the component dialog, configure these fields:

GENERAL

  • Brand label: the brand label of the section;
  • Legal Notes title: the legal notes title of the section;
  • Legal Notes description: the legal notes description of the section;

APP LINKS

  • Title: the title of the section;
  • Build social link using: choose the page list type:
    • Fixed List: allow the selection of the single pages
    • Child pages: show all the children page of a hub page
  • Option for Child Pages: the path of the container page;

 

In order to modify the icon and the name of the pages shown in the footer follow these steps:

  1. Select the linked page
  2. Click “View Properties”
  3. In the “Navigation” tab configure these fields:

    • Page Logo Url: the logo image of app page;

    SOCIAL LINKS

    • Title: the title of the section;
    • Build social link using: choose the page list type:
      • Fixed List: allow the selection of the single pages
      • Child pages: show all the children page of a hub page
    • Option for Child Pages: the path of the container page;
