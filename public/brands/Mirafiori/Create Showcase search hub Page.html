2. HOW TO CREATE THE SHOWCASE SEARCH HUB PAGE

In order to create the Showcase search Hub Page, follow these steps:

  1. Select the language page (e.g. mirafiorioutlet/mirafiorioutlet-it/it)
  2. Click: Create > Create Page
  3. Choose “Showcase Search Hub Page Template” as page layout
  4. Click Next
  5. Insert page data (for example, title can be “last minute”)

  6. Click “Create”;

  7. Click “Done” in the opened popup

3. HOW TO CONFIGURE SHOWCASE SEARCH HUB PAGE

In order to configure the Showcase search Hub page follow these steps:

  1. Select the language page (e.g. mirafiorioutlet/mirafiorioutlet-it/it)
  2. Click “Properties”
  3. In the “SEARCH HUB” tab configure these fields:

Banner

  • Background image:the background image path of banner section;
  • Title:the title of banner section;
  • Subtitle:the subtitle of banner section;
  • Description: the description of banner section;

Search Settings

  • Showcase lookup reference:
  • Vehicle Type: the vehicle of search section (Not New, New, Used or Km0);
  • Brand:the brand of search section (None, Fiat, Fiat Pro, Alfa Romeo, Jeep, Abarth or Lancia);

Enable selector

  • Selector:the selector of selector section (none, City, Brand and Model, or Brand, Model and City);

4. HOW TO CONFIGURE THE SHOWCASE LOOKUP PAGE

In order to configure the Showcase lookup page follow these steps:

  1. Select the Showcase lookup page (for example “last minute”) (e.g. mirafiorioutlet/mirafiorioutlet-it/it/showcase-lookup/last-minute)
  2. Click “Properties”
  3. In the “SHOWCASE LOOKUP” tab configure these fields:

Filter

  • TAG:the tag of the Showcase lookup page (Last minute, Premium, Most visited, Eco or Neo-p);

General

  • IconClass:the icon of the Showcase lookup page;
