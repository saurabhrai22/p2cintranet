SCOPE

The goal of this document is to provide the basic tips in order to create Home page in the Mirafiori Outlet portal.


HOW TO CREATE THE HOME PAGE

In order to create the Home Page, follow these steps:

  1. Select the language page (e.g. mirafiorioutlet/mirafiorioutlet-it/it)
  2. Click: Create > Create Page
  3. Choose “Mirafiori Outlet Editorial Template” as page layout
  4. Click Next
  5. Insert page data (a title can be “Home”)
  1. Click “Create”;
  2. Click “Done” in the opened popup

HOW TO CONFIGURE THE SEARCH

In order to configure the Search follow these steps:

  1. Select the “it” page (e.g. mirafiorioutlet/mirafiorioutlet -it/it)
  2. Click “Properties”
  3. In the “I18N” tab configure these fields:

Basic configuration

  • Standard search Hub Page: the standard search hub page;

 

In order to complete the configuration of the Search follow these steps:

  1. For example, select the “last minute” page (e.g. mirafiorioutlet/mirafiorioutlet -it/it/ricercar-auto-usate)
  2. Click “Properties”
  3. In the “SEARCH FORM” tab configure these fields:

Search Settings

  • Choose the brand: the choose brand label of the section;
  • Choose the model: the choose model label of the section;
  • Search button: the search button label of section;
  • Results: the results label of the section;
  • City: the city label of the section;

HOW TO CONFIGURE THE SHOWCASES

In order to configure the Showcases follow these steps:

  1. Select the “it” page (e.g. mirafiorioutlet/mirafiorioutlet -it/it)
  2. Click “Properties”
  3. In the “I18N” tab configure these fields:

Basic configuration

  • Showcase lookup Page: the showcase lookup page;

 

In order to complete the configuration of the Showcases follow these steps:

  1. For example, select the “last minute” page (e.g. mirafiorioutlet/mirafiorioutlet -it/it/showcase-lookup)
  2. Click “Properties”
  3. In the “SHOWCASE LOOKUP” tab configure these fields:

 

General

  • IconClass: the icon of Last minute;

 

Visibility

  • Show in home page: check this field to display “Last minute” showcase lookup in home page;

 

Home page

  • Image: the image of showcase in home page;

HOW TO CONFIGURE THE VIRTUAL ASSISTANT

In order to configure the Virtual assistant follow these steps:

  1. Select the “it” page (e.g. mirafiorioutlet/mirafiorioutlet -it/it)
  2. Click “Properties”
  3. In the “I18N” tab configure these fields:

Assistant Label Settings

  • Private Area: the private area label of the section;
  • Compare: the compare label of the section;
  • Vehicle Details: the vehicle detail label of the section;
  • Double Promo Home: the double promo home label of the section;
  • Home: the home label of the section;
  • Home Showcase Visible: the home showcase visible label of the section;
  • Search Results: the search results label of the section;
  • Get Recommendation: the get recommendation label of the section;
