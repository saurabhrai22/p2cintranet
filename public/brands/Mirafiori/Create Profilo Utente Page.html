1. SCOPE

    The goal of this document is to provide the basic tips in order to create Profilo utente page in the Mirafiori Outlet portal.

2. HOW TO CREATE THE PROFILO UTENTE PAGE

    In order to create the Profilo utente Page, follow these steps:

  1. Select the language page (e.g. mirafiorioutlet/mirafiorioutlet-it/it)
  2. Click: Create Page
  3. Choose “Mirafiori Outlet User Profile Template” as page layout
  4. Click Next
  5. Insert page data (a title can be “profile-utente”

  6. Click “Create”;
  7. Click “Done” in the opened popup

3. HOW TO CONFIGURE THE PROFILO UTENTE PAGE

    In order to configure the Profilo utente page follow these steps:

  1. Select the “registration” page (e.g. mirafiorioutlet/mirafiorioutlet-it/it/profilo-utente)
  2. Click “Properties”
  3. In the “USER PROFILE” tab configure these fields:
