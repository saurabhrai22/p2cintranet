SCOPE

    The goal of this document is to provide the basic tips in order to create Search Results page in the Mirafiori Outlet portal.

HOW TO CREATE THE SEARCH RESULTS PAGE

    In order to create the Search results Page, follow these steps:

  1. Select the language page (e.g. mirafiorioutlet/mirafiorioutlet-it/it )
  2. Click: Create > Create Page
  3. Choose “Standard Search Hub Page Template” as page layout
  4. Click Next
  5. Insert page data (a title can be “Ricerca auto usate”)
  6. Click “Create”;
  7. Click “Done” in the opened popup
