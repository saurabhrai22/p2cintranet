1. SCOPE

  • The goal of this document is to provide the basic tips in order to create the Dati societari page in the Mirafiori Outlet portal.

2. HOW TO CREATE THE CONCESSIONARI PAGE

In order to create the Dati societari Page, follow these steps:

  1. Select the language page (e.g. mirafiorioutlet/mirafiorioutlet-it/it/)
  2. Click: Create > Create Page
  3. Choose “Mirafiori Outlet Editorial Template” as page layout
  4. Click Next
  5. Insert page data (a title can be “Dati societari”)

    6. Click “Create”;
    7. Click “Done” in the opened popup

3.HOW TO CONFIGURE CONCESSIONARI PAGE

In order to configure the Concessionari page follow these steps:

  1. Select the “it” page (e.g. mirafiorioutlet/mirafiorioutlet -it/it)
  2. Click “Properties”
  3. In the “I18N” tab configure these fields:

  • headline: the headline label of the section;
  • cityInput: the city label of the section;
  • searchCTA: the search cta label of the section;

4. HOW TO CONFIGURE THE DEALER LOCATOR COMPONENT

In order to configure the “Dealer locator” component into Concessionari page, open the page in “Edit Mode” and see related document “AEM – Mirafiori Outlet – Components Catalog”.
