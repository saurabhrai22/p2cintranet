How to Create Home Page

  • In order to create the Home Page, follow these steps:
    1. Select the language page (e.g. fiatprofessional/ fiatprofessional-it/it/).
    2. Click: Create > Create Page.
    3. Choose “Fiat Pro Editorial Template”as page layout.
    4. Click: Next.
    5. Insert page data (a title can be “Home”.)
  1. Click: "Create";
  2. Click: "Done" ” in the opened popup.

Tags on Model Pages

On each model version node properties (the child node), of the models available on reference market, in the Need based tool tab ( screenshot below), you need to add the tags on all 6 filters

These tags are the same ones for all markets and you can copy them from .IT market for all 6 filters.

How to Configure Equipment Component

  1. Select the equipment page under hub equipments (e.g. fiatprofessional/ fiatprofessional-it/it/Hub equipments/ducato).
  2. Click: "Properties".
  3. In the “HUB NAVIGATION tab configure these fields:
  • Layout: the layout for the section (in this case for example, select “Hub equipments”).
  • Title: the title for the section.
  • Description: the description for the section.
  • CTA label: the label for the cta.
  • Alt image: the alt image field.
  • Image: the image for the tile.
  • Icon x: the icon for the top feature.
  • Title x: the title for the top feature.
  • Subtitle x: the subtitle for the top feature.
