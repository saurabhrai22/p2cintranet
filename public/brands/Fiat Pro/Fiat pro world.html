HOW TO CREATE THE FIAT PRO WORLD PAGE

In order to create the Financial services Page, follow these steps:

    1. Select the language page (e.g. fiatprofessional/ fiatprofessional-it/it/).
    2. Click: Create > Create Page.
    3. Choose “Fiat Pro Editorial Template” as page layout.
    4. Click Next.
    5. Insert page data (a title can be “Financial services”).
    1. Click “Create”.
    2. Click “Done” in the opened popup.

HOW TO CONFIGURE THE NEWS CAROUSEL COMPONENT

In order to add News Carousel component into the page follow these steps:
  1. Click toggle side panel on the page ribbon
  2. Select “Components” tab
  3. Choose “News Carousel” component
  4. Drag and drop the component in the page.
  5. Click on the “Configure” icon in order to set the information of the component.
  6. In the dialog, configure these fields:

GENERAL

  • Title HTML tag: the HTML tag for the title in the section
  • Title: the title for the section
  • Subtitle: the subtitle for the section
  • News detail page: the page for the tile

HOW TO CONFIGURE THE NEWS SLIDES IN THE NEWS CAROUSEL COMPONENT

In order to configure the News page follow these steps:

    1. Select the “it” page (e.g. fiatprofessional/ fiatprofessional-it/it/fiat pro world/news).
    2. Click: "Properties".
    3. In the “News” tab configure these fields:

News Basic

  • News tags: the tags of the page;
  • Date: the date of the news;
  • Display date: the date of the news displayed on the page;
  • Title: the title of the page;

News content slider

  • Image (desktop): the path of the image shown in desktop view;
  • Image (smartphone landscape): the path of the image shown in smartphone landscape view;
  • Image (smartphone): the path of the image shown in smartphone view;
FIND MORE LABEL CONFIGURATION
    1. Select the language page (e.g. fiatprofessional/ fiatprofessional-it/it/).
    2. Click: "Properties".
    3. In the “I18N” tab configure this field:
  • Find more Label: the find more label;
