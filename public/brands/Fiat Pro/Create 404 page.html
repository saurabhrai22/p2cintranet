HOW TO CREATE THE 404 PAGE

In order to create the 404 Page, follow these steps:

  1. Select the language page (e.g. fiatprofessional/ fiatprofessional-it/it)
  2. Click: Create > Create Page
  3. Choose “Fiat Pro Editorial Template” as page layout
  4. Click Next
  5. Insert page data (a title can be “404”)
  1. Click “Create”;
  2. Click “Done” in the opened popup

HOW TO CONFIGURE THE 404 COMPONENT

In order to add the “404” component into the page follow these steps:

  1. Click toggle side panel on the page ribbon
  2. Select “Components” tab
  3. Choose “404” component
  4. Drag and drop the component in the page.
  5. Click on the “Configure” icon in order to set the information of the component.
  6. In the dialog, configure these fields:

GENERAL

  • Title: the title of the section
  • Subtitle: the subtitle of the section
  • CTA label: the CTA label for the section
  • CTA linked page: the CTA linked page for the section

IMAGE

  • Alt image the image of the section
  • Image (desktop): the desktop image for the section
  • Image (tablet): the tablet image for the section
  • Image (smartphone landscape): the smartphone landscape image for the section
  • Image (smartphone): the smartphone image for the section
