Footer CTA

In order to configure the Footer CTA section, follow these steps:

  1. Select the Footer CTA Component.
  2. Click: "Edit" button.
  3. In the component dialog, configure these fields:
  • Build social link using: choose the page list type:
    • Fixed List: allow the selection of the single pages.
    • Children: show all the children page of a hub page.
  • Option for Fixed List
    • Pages: the path of the page.

In order to modify the icon and the name of the pages shown in the footer cta section follow these steps:

  1. Select the linked page.
  2. Click: "View Properties".
  3. In the "Navigation" tab configure these fields:
  • Navigation Title: the name of the page shown in the header.
  • CSS Class: the icon class.
