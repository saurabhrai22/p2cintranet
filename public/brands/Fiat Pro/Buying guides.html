HOW TO CREATE THE BUYING GUIDES PAGE

In order to create the Buying guides Page, follow these steps:

  1. Select the language page (e.g. fiatprofessional/ fiatprofessional-it/it/).
  2. Click: Create > Create Page.
  3. Choose “Fiat Pro Editorial Template” as page layout.
  4. Click Next.
  5. Insert page data (a title can be “Equipments Detail”).
  6. Click “Create”.
  7. Click “Done” in the opened popup.

HOW TO CREATE THE CHILD PAGE FOR THE BUYING GUIDES

You have to create the child page in order to use the Buying guides Page.
To create this child page, follow this steps:

  1. Select the Buying guides page (e.g. fiatprofessional/ fiatprofessional-it/it/Buying guides).
  2. Click: Create > Create Page.
  3. Choose“External/Redirect Template” s page layout
  4. Click Next.
  5. Insert page data (a title can be Financial calculator)
  6. Click “Create”.
  7. Click “Done” in the opened popup.

HOW TO CREATE THE CHILD PAGE FOR THE BUYING GUIDES

  1. Select the child page under Buying guides (e.g. fiatprofessional/ fiatprofessional-it/it/Buying guides/financial calculator).
  2. Click: “Properties”.
  3. In the “HUB NAVIGATION” tab configure these fields:

 

  • Layout:the layout for the section (in this case for example, select “Hub buying guides”)
  • Title:the title for the section
  • Description:the description for the section
  • CTA label:the label for the cta
  • Alt image: the alt image field
  • Image:the image for the tile

HOW TO CONFIGURE THE HUB NAVIGATION COMPONENT

In order to add Hub navigation component into the page follow these steps:

  1. Click toggle side panel on the page ribbon
  2. Select “Components” tab
  3. Choose “Hub navigation” component
  4. Drag and drop the component in the page.
  5. Click on the “Configure” icon in order to set the information of the component.
  6. In the dialog, configure these fields:

GENERAL

  • Title: the title of the section
  • Dimension title: : the dimension for the title (can be h1 or h2)
  • Subtitle: : the subtitle of the section
  • Background color: the background color of the section (can be white or grey)
  • Style: the style for the section (can be 2 o 3 columns)

TILES

 

  • Pages: the linked pages for the section (in this field, you have to add the pages that have “External/redirect” template in order to see the tiles correctly)
