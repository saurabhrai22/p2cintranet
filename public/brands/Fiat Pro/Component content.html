HOW TO CREATE THE COMPONENT CONTENT PAGE

In order to create the Component content Page, follow these steps:

  1. Select the language page (e.g. fiatprofessional/ fiatprofessional-it/it/).
  2. Click: Create > Create Page.
  3. Choose “Grouping Page Template” as page layout.
  4. Click Next.
  5. Insert page data (a title can be “Component content”).
  6. Click “Create”.
  7. Click “Done” in the opened popup.

HOW TO CREATE THE CHILD PAGE FOR THE COMPONENT CONTENT

You have to create the child page in order to use the Component content Page.
To create this child page, follow this steps:

  1. Select the language page (e.g. fiatprofessional/ fiatprofessional-it/it/Component content).
  2. Click: Create > Create Page.
  3. Choose “Vehicle Equipment Template” as page layout.
  4. Click Next.
  5. Insert page data (a title can be “Trasporto Persone”).
  6. Click “Create”.
  7. Click “Done” in the opened popup.

HOW TO CREATE THE CHILD PAGE FOR THE TRASPORTO PERSONE

You have to create the child page in order to use the Trasporto persone Page. To create this child page, follow this steps:

  1. Select the Component content page (e.g. fiatprofessional/ fiatprofessional-it/it/Component content/Trasporto Persone).
  2. Click: Create > Create Page.
  3. Choose “Detail Equipment Template” as page layout.
  4. Click Next.
  5. Insert page data (a title can be “1 Allestimento Fiat Pro”).
  6. Click “Create”.
  7. Click “Done” in the opened popup.

HOW TO CREATE THE CHILD PAGE FOR THE TRASPORTO PERSONE

  1. Select the child page under Hub mopar services( fiatprofessional/ fiatprofessional-it/it/Component content/Trasporto Persone/1 Allestimento Fiat Pro).
  2. Click: "Properties".
  3. In the “DETAIL EQUIPMENT” tab configure these fields:

General

  • Title:the title for the section
  • CTA Label:the label for the CTA button
  • CTA linked page:the path of the linked page of CTA

Vehicle X

  • Name:the title for the section
  • Description:the label for the CTA button
  • Image (desktop):the path of the image shown in desktop view
  • Image (smartphone):the path of the image shown in smartphone view
