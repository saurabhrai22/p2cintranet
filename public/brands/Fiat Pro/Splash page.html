HOW TO CREATE THE SPLASH PAGE

In order to create the Splash Page, follow these steps:

  1. Select the language page (e.g. fiatprofessional/ fiatprofessional-be).
  2. Click: Create > Create Page.
  3. Choose “Fiat Pro Website Splash Page Template” as page layout.
  4. Click Next.
  5. Insert page data (a title can be “splash-page”).
  • Click "Create".
  • Click “Done” in the opened popup.
    • Importantsplash page can be created when a country have multiple languages. For example, be have nl and fr

    HOW TO CONFIGURE THE SPLASH COMPONENT

    In order to configure the Need based tool page follow these steps:

    1. Select the splash page (e.g. fiatprofessional/ fiatprofessional-be/splash-page).
    2. Click: "“Properties”".
    3. In the “SPLASH PAGE” tab configure these fields:
    General
    • Title:the title of the section

     

    Image
    • Image (desktop):the desktop background image for the section
    • Image (tablet):the tablet background image for the section
    • Image (smartphone landscape):the smartphone landscape background image for the section
    • Image (smartphone):the smartphone background image for the section

     

    Language 1
    • CTA label:the label of cta
    • CTA linked page:the linked page for the slide
    • Data language:the label of data language

     

    Language 2
    • CTA label:the label of cta
    • CTA linked page:the linked page for the slide
    • Data language:the label of data language

     

    Language 3
    • CTA label:the label of cta
    • CTA linked page:the linked page for the slide
    • Data language:the label of data language
