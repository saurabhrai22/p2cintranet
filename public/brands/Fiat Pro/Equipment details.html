HOW TO CREATE THE EQUIPMENTS DETAIL PAGE

In order to create the Equipments Detail Page, follow these steps:

  1. Select the language page (e.g. fiatprofessional/ fiatprofessional-it/it/).
  2. Click: Create > Create Page.
  3. Choose “Fiat Pro Editorial Template” as page layout.
  4. Click Next.
  5. Insert page data (a title can be “Equipments Detail”).
  • Click “Create”.
  • Click “Done” in the opened popup.

HOW TO CONFIGURE EQUIPMENT DETAIL PAGE

In order to configure the Equipment Details page follow these steps:

  1. Select the “it” page (e.g. fiatprofessional/ fiatprofessional-it/it).
  2. Click “Properties”.
  3. In the “i18N” tab configure this field:

DEALER LOCATOR

  • bodyshop: the label of body shop section;
  • dealer: the label of dealer section;
  • garage: the label of garage section;
  • camper: the label of camper section;
  • specialist: the label of specialist section;
  • organized: the label of organized section;
  • website: the label of website section;
  • measure : the label of measure section;
  • Find dealer placeholder: the label of find dealer placeholder section;
  • Filter by services : the label of find services section;
  • Services label: the label of services section;

HOW TO CONFIGURE THE EDITORIAL COVER IMAGE COMPONENT

In order to add the “Editorial Cover Image” component into the page follow these steps:

  1. Click toggle side panel on the page ribbon
  2. Select “Components” tab
  3. Choose “Editorial Cover Image” component
  4. Drag and drop the component in the page.
  5. Click on the “Configure” icon in order to set the information of the component.
  6. In the dialog, configure these fields:

 

  • Title: the title of the section
  • Description: the description of the section
  • Alt Image: the message that appear if image inserted is not longer present
  • Image: the image of the section
  • Alt Image: the message that appear if background image inserted is not longer present
  • Background Image: the background image of the section

HOW TO CONFIGURE DEALER LOCATOR VIEWER IN EQUIPMENT DETAIL PAGE

In order to add the “Dealer Locator Viewer” component into the page follow these steps:

  1. Click toggle side panel on the page ribbon
  2. Select “Components” tab
  3. Choose “Dealer Locator Viewer” component
  4. Drag and drop the component in the page.
  5. Click on the “Configure” icon in order to set the information of the component.
  6. In the dialog, configure these fields:
General
  • Title: the title of the section;
  • Title tag: the style of title text (H1,H2 or H3);
  • Subtitle: the subtitle of the section;
  • Included page path: the path of the dealer locator page (the path should be for example: /content/fiatprofessional/fiatprofessional-it/it/trova-concessionarie);
