HOW TO CREATE OURCARS HUB PAGE

The Ourcars HUB Pages is a child of Website Language Configuration page (e.g. fiatprofessional/ fiatprofessional-it/it) In order to create the page Ourcars HUB page follow these steps:

    1. Select the language page (e.g. fiatprofessional/ fiatprofessional-it/it).
    2. Click: Create > Create Page.
    3. Choose “Ourcars HUB Template” as page layout.
    4. Click Next.
    5. Insert page data (a title can be “Models”).
  1. Click “Create”.
  2. Click “Done” in the opened popup.

HOW TO CONFIGURE FILTER SECTION IN THE OURCARS HUB PAGE

In order to configure the Filter section into Ourcars HUB, follow this steps:

    1. Open the properties of Ourcars HUB page, by selecting this page and clicking on "Properties".

 

 

    1. Select the Ourcars HUB page (e.g. fiatprofessional/ fiatprofessional-it/it/Models).
    2. Click "Properties".
    3. In the "Filter"tab configure these fields:

 

  1. Mobile titlethe title label visible in mobile version
  2. ALL the placeholder label for the filter section

HOW TO CONFIGURE THE MODEL SECTION OF THE FILTER IN THE OURCARS HUB PAGE

The Model section is showed in the ourcars HUB when the equipment pages or the model pages are tagged. You also need to create the “ Ourcars detail ” pages Follow these steps to configure it:

  1. Select the equipment or the model page you want to tag 
    (e.g.  fiatprofessional/ fiatprofessional-it/it/Gamma Fiat Professional/ fullback cabina doppia or fiatprofessional/ fiatprofessional-it/it/Gamma Fiat Professional/ fullback cabina doppia/sx )

    (If the tag is placed inside a model page you can see the first child of it on Ourcars HUB, otherwise if the tag is placed inside an equipment page you can see it on Ourcars HUB page)

HOW TO TAG THE EQUIPMENT PAGE

The Model section is showed in the ourcars HUB when the equipment pages or the model pages are tagged. You also need to create the “Ourcars detail” pages Follow these steps to configure it:

  1. Select the equipment page (e.g. . fiatprofessional/ fiatprofessional-it/it/Gamma Fiat Professional/ fullback cabina doppia/sx)).
  2. Click "Properties".
  3. In the "OUR CAR"tab configure the tag field
  4. Choose the tag you want to use (e.g. . fiatprofessional/ourcar/category/pick-up)).

HOW TO CONFIGURE THE TAGGED EQUIPMENT PAGE

  1. In the "OUR CAR"tab configure the tag field
  • Titlethe title label for the box
  • Single promotion label the single promotion label for the box
  • Choose the tag you want to use (e.g. . fiatprofessional/ourcar/category/pick-up)).

HOW TO CREATE A SPECIFIC NEEDS SETTINGS

This component is provided by “Ourcars HUB Template”, configure the fields in dialog as shown below:

GENERAL

  • Title: the specific needs title

TAB X

  • Title the title for the tab x
  • Description: the description for the tab x
  • CTA label: the CTA label for the tab x
  • CTA linked page: the CTA linked page for the tab x
