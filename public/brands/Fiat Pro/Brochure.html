HOW TO CREATE THE BROCHURE PAGE

In order to create the Brochure Page, follow these steps:

  1. Select the language page (e.g. fiatprofessional/ fiatprofessional-it/it)
  2. Click: Create > Create Page
  3. Choose “Fiat Pro Editorial Template” as page layout
  4. Click Next
  5. Insert page data (a title can be “Direct-brochure”)
  1. Click “Create”;
  2. Click “Done” in the opened popup

HOW TO CONFIGURE THE BROCHURE PAGE

  1. Select the equipment page under Models Form Lookup (e.g. fiatprofessional/ fiatprofessional-it/it/models-form-lookup/fullback-cabina-doppia)
  2. Click“Properties”
  3. In the “MODEL DATA” figure these fields:

Basic configuration

  • Alt image:the alt image field;
  • Image (desktop):the desktop image for the tile;
  • Image (smartphone):the smartphone image for the tile;
  • Thank you model CTA:the page for the tile;

Brochure

  • Brochure:the pdf for the brochure;
  • Catalog:the pdf for the catalog;
  • Specifications:the pdf for the specifications;
  • Pricelist: the pdf for the price list;

Configurator

  • Configurator Page:the page for the tile;

HOW TO CONFIGURE THE TITLE COMPONENT

In order to add the “Editorial Info Tile” component into the page follow these steps:

  1. Click toggle side panel on the page ribbon
  2. Select “Components” tab
  3. Choose “Editorial Info Tile” component
  4. Drag and drop the component in the page.
  5. Click on the “Configure” icon in order to set the information of the component.
  6. In the dialog, configure these fields:

GENERAL

  • Title: the title for the section
  • Title HTML tag: the HTML tag for the title in the section
  • Description: the description for the section
  • Abstract:the abstract for the section
  • Show highlight: the checkbox for show/hide the highlight over the title in the section

HOW TO CONFIGURE THE BROCHURE AND CONFIGURATOR HUB COMPONENT

In order to add the “Brochure and Configurator Hub” component into the page follow these steps:

  1. Click toggle side panel on the page ribbon
  2. Select “Components” tab
  3. Choose “Brochure and Configurator Hub” component
  4. Drag and drop the component in the page.
  5. Click on the “Configure” icon in order to set the information of the component.
  6. In the dialog, configure these fields:

GENERAL

  • Select Type: the layout for the page (Brouchure or Configurator);
  • Models Form Lookup page: the page for the tile;

BROCHURE AND CONFIGURATOR

Brochure

  • Title: the title of the Brochure section;
  • Subtitle: the subtitle of the section;
  • Brochure CTA Label 1: the first CTA label of the section;
  • Brochure CTA Label 2: the second CTA label of the section;
  • Brochure CTA Label 3: the third CTA label of the section;
  • Brochure CTA Label 4: the fourth CTA label of the section;

Configurator

  • Title: the title of the Configurator section;
  • Configurator CTA Label: the CTA label of the section;
