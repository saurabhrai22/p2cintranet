Top Header Section

In order to configure the top header section, follow these steps:

  1. Select the language node
  2. Click: "View Properties"button
  3. In the “Navigation” configure these fields:

 

  • Top Header Call to Action: the path of the pages linked in the top header; click “Add field” to add another page
  • Top Header My Fiat Professional:the path of my fiat professional page
  • Top Header Select Language Menu Label:the language label

Link Section

In order to configure the link section of the header follow these steps:

  1. Select the Header component
  2. Click on “Edit” button
  3. In the component dialog configure these fields:
  • Build List Using: choose the page list type:
  • Fixed List: allow the selection of the single pages
  • Children: show all the children page of an hub page
  • Option for Fixed List
  • Title: the title of the list
  • Pages: the path of the page
