In order to create the Search Page, follow these steps:

  1. Select the language page (e.g. fiatprofessional/ fiatprofessional-it/it/).
  2. Click: Create > Create Page.
  3. Choose “Fiat Pro Editorial Template” as page layout.
  4. Click Next.
  5. Insert page data (a title can be “Cerca”).
  6. Click "Create".
  7. Click “Done” in the opened popup.

HOW TO CONFIGURE THE SEARCH COMPONENT

In order to add the Search component into the page follow these steps:

  1. Click toggle side panel on the page ribbon
  2. Select “Components” tab
  3. Choose ““Search”” component
  4. Drag and drop the component in the page.
  5. Click on the “Configure” icon in order to set the information of the component.
  6. In the dialog, configure these fields:

LABEL

  • resoultfound: the resoult found label
  • findmore: the found more label; *
*find more label is displayed only if there are result for the search

HOW TO CONFIGURE THE SEARCH PAGE

In order to configure the Search page follow these steps:

  1. Select the “it” page (e.g. fiatprofessional/ fiatprofessional-it/it/).
  2. Click: "“Properties”".
  3. In the ““Navigation”” tab configure these fields:
  • Search Landing page:the page when the search are displayed
  • Search Label:the label of search section for the page
