HOW TO CONFIGURE A FORM

In order to configure a Form follow these steps:

  1. Select the “it” page (e.g. fiatprofessional/ fiatprofessional-it/it)
  2. Click: "Properties".
  3. In the "Form Standard Reference" tab configure these fields:

Test Drive

  • Test drive Models: the test drive models page;
  • Test Drive page: the test drive page;


Get a quote

  • Get a quote Models: the get a quote models page;
  • Get a quote page: the get a quote page;


Stock locator

  • Pc Models: the pc models page;
  • Pc page: the pc page;


Request info

  • Request info Models: the request info models page;
  • Request info page: the request info page;


Get a brochure

  • Request Brochure Models: the request brochure models page;
  • Request Brochure page: the request brochure page;
  • Request brochure digital page: the request brochure digital page;
  • Enable modal choice: the checkbox to able the choice between brochure page or brochure digital page; *

Eloqua form Tab labels

  • Tab Name form-TD: the tab name label form for the test drive;
  • Tab Name form-GAQ: the tab name label form for the get a quote;
  • Tab Name form-PC: the tab name label form for pc;
  • Tab Name form-RI: the tab name label form for the request information;

In order to configure a Form follow these steps:

  1. Select the language page (e.g. fiatprofessional/ fiatprofessional-it/it)
  2. Click: "Create > Create Page".
  3. Choose "External/Redirect Template" as page layout.
  4. Click "Next".
  5. Insert page data (a title can be “brochure-modal”)
  1. Click "Create";
  2. Click "Done" in the opened popup.

HOW TO CONFIGURE THE ELOQUA STANDARD FORMS COMPONENT

In order to add the “Eloqua Standard Forms” slider component into the page follow these steps:

  1. Click toggle side panel on the page ribbon
  2. Select “Components” tab
  3. Choose “Eloqua Standard Forms” component
  4. Drag and drop the component in the page.
  5. Click on the “Configure” icon in order to set the information of the component.
  6. In the dialog, configure these fields:

General

  • Title: the title of the section;
  • Description: the description of the section;
  • Select the CTA to be Show: the CTA showed;
  • Select the default CTA: the default CTA;
