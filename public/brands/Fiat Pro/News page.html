In order to create the Equipments Detail Page, follow these steps:

  1. Select the language page (e.g. fiatprofessional/ fiatprofessional-it/it/fiat professional world).

  4. Click Next.
  5. Insert page data (a title can be ““News””).

  2. Click “Done” in the opened popup.

HOW TO CONFIGURE THE EDITORIAL INFO TILE COMPONENT

In order to add the “Editorial Info Tile” component into the page follow these steps:

  1. Click toggle side panel on the page ribbon
  2. Select “Components” tab
  3. Choose “Editorial Info Tile” component
  4. Drag and drop the component in the page.
  5. Click on the “Configure” icon in order to set the information of the component.
  6. In the dialog, configure these fields:

 

GENERAL



Editorial
  • Subtitle: the subtitle of the section;
  • Title: the title of the section;
  • Description: the description of the section;
  • CTA Label: the label for CTA button;
  • CTA linked page: the path of the linked page of CTA;
  • Background color: the color of background (White or Grey);
  • CSS padding: the CSS padding for the section (by default, leaving this field blank, the value is padding-top-40)

Image
  • Alt image: the message that appear if image inserted is not longer present;
  • Image (desktop): the path of the image shown in desktop view;
  • Image (smartphone): the path of the image shown in smartphone view;

HOW TO CONFIGURE THE FIAT PRO WALL COMPONENT

In order to add Fiat pro wall component into the page follow these steps:

  1. Click toggle side panel on the page ribbon
  2. Select “Components” tab
  3. Choose “Fiat pro wall” component
  4. Drag and drop the component in the page.
  5. Click on the “Configure” icon in order to set the information of the component:
  6. In the dialog, configure these fields

 

GENERAL


  • Filter title: the filter title for the section
  • News label:the label for the news in the section
  • Search placeholder: the search placeholder in the section
  • News number: the number of news visible in the section
  • Tag: the tag for the section (if this field is blank you see all of the tags used in the news detail page, otherwise you see the inserted tag)
  • CTA label: the label for the cta in this section
  • Show more label: : the label for the show more button, it appear if the number of news if bigger than the news number value

FIXED TILE

  • Alt image: the alt image for the fixed tile section
  • Image (desktop): the desktop image for the fixed tile section
  • Image (smartphone landscape): the smartphone landscape image for the fixed tile section
  • Image (smartphone): the smartphone image for the fixed tile section
  • CTA Label: the label for the CTA in the fixed tile section
  • CTA linked page: the linked page for the CTA in the fixed tile section
