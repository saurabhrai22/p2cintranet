How to Create Hub Mopar Services Page

In order to create the Hub Mopar Services Page, follow these steps:

  1. Select the language page (e.g. fiatprofessional/ fiatprofessional-it/it/).
  2. Click: Create > Create Page.
  3. Choose “Fiat Pro Editorial Template”as page layout.
  4. Click: Next.
  5. Insert page data (a title can be “Hub Mopar Services”.)
  1. Click: "Create";
  2. Click: "Done" ” in the opened popup.

How to create child page for the hub mopar services

You have to create the child page in order to use the HUB mopar services Page.

In order to create the child Page, follow these steps:

  1. Select the Hub equipments page (e.g. fiatprofessional/ fiatprofessional-it/it/).
  2. Click: Create > Create Page.
  3. Choose “External/Redirect Template”as page layout.
  4. Click: Next.
  5. Insert page data (a title can be “My Vehicle.”.)
  1. Click: "Create";
  2. Click: "Done" ” in the opened popup.

How to configure child page

  1. Select the Hub equipments page (e.g. fiatprofessional/ fiatprofessional-it/it/Hub mopar services/my Vehicle).
  2. Click: "Properties".
  3. In the “HUB NAVIGATION tab configure these fields:
  • Layout: the layout for the section (in this case for example, select “Hub equipments”).
  • Title: the title for the section.
  • Description: the description for the section.
  • CTA label: the label for the cta.
  • Alt image: the alt image field.
  • Image: the image for the tile.
  • Icon x: the icon for the top feature.
  • Title x: the title for the top feature.
  • Subtitle x: the subtitle for the top feature.
