HOW TO CREATE THE END SERIES PAGE

In order to create the End series Page, follow these steps:

  1. Select the language page (e.g. fiatprofessional/ fiatprofessional-it/it)
  2. Click: Create > Create Page
  3. Choose “Fiat Pro Editorial Template” as page layout
  4. Click Next
  5. Insert page data (a title can be “End series”))
  6. Click “Create”;
  7. Click “Done” in the opened popup

HOW TO CONFIGURE THE TITLE COMPONENT

In order to add the “Editorial Info Tile” component into the page follow these steps:

  1. Click toggle side panel on the page ribbon
  2. Select “Components” tab
  3. Choose “Editorial Info Tile” component
  4. Drag and drop the component in the page.
  5. Click on the “Configure” icon in order to set the information of the component.
  6. In the dialog, configure these fields:

GENERAL

Editorial
  • Subtitle: the subtitle of the section;
  • Title: the title of the section;
  • Description: the description of the section;
  • CTA Label: the label for CTA button;
  • CTA linked page: the path of the linked page of CTA;
  • Background color: the color of background (White or Grey);
  • CSS padding: the CSS padding for the section (by default, leaving this field blank, the value is padding-top-40)
Image
  • Alt image: the message that appear if image inserted is not longer present;
  • Image (desktop): the path of the image shown in desktop view;
  • Image (smartphone): the path of the image shown in smartphone view;
