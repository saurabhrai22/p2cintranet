How to Create Model Container Page

All the Model Pages are children of a container page

In order to create the Hub Mopar Services Page, follow these steps:

  1. Select the language page (e.g. fiatprofessional/ fiatprofessional-it/it/).
  2. Click: Create > Create Page.
  3. Choose “Grouping Page Template”as page layout.
  4. Click: Next.
  5. Insert page data (a title can be ““Gamma Fiat Professional””.)
  1. Click: "Create";
  2. Click: "Done" ” in the opened popup.

How to create Model Page

In order to create the Model Page, follow these steps:

  1. Select the Hub equipments page (e.g. fiatprofessional/ fiatprofessional-it/it/).
  2. Click: Create > Create Page.
  3. Choose “Model Template”as page layout.
  4. Click: Next.
  5. Insert page data (a title can be “fullback cabina doppia.”.)
  1. Click: "Create";
  2. Click: "Done" ” in the opened popup.

How to Configure Compare Table in Model Page

The Compare Table is provided by the Model Page template; follow these steps to configure it:

  1. Select the language page (e.g. fiatprofessional/ fiatprofessional-it/it/Gamma Fiat Professional/fullback cabina doppia).
  2. Click: "Properties".
  3. In the “Model Properties” tab configure these fields:
  • CPS Anchor-Tag: the CPS anchor tag.
  • Configurator page: the path of the configurator page.
  • Current Model: the path of the current model page.
  • Performance Characteristics Table Key List Order
    • Key List Order: the CPS anchor tag.
    • Add field: click to add another characteristic name label.

In this table will be compare the equipments of the model.

HOW TO CREATE THE EQUIPMENT PAGE
In order to create an equipment for the model follow these steps:
  1. Select the model page (e.g. fiatprofessional/ fiatprofessional-it/it/Gamma Fiat Professional/fullback cabina doppia).
  2. Click: Create > Create Page.
  3. Choose “Model Template”as page layout.
  4. Click: Next.
  5. Insert page data (a title can be “sx”.)
  • Model name: the name of the model equipment.
  • Alt image: : the message that appear if image inserted is not longer present.
  • Model Image (Desktop): the path of the image shown in desktop view.
  • Model Image (Smartphone Landscape):the path of the image shown in smartphone Landscape view
  • Performance and Characteristics Table
    • Value List Order: the value of the characteristic.
    • Add field: click to add another characteristic value.
