HOW TO CREATE THE CAMPER PAGE

In order to create the 404 Page, follow these steps:

  1. Select the language page (e.g. fiatprofessional/ fiatprofessional-it/it)
  2. Click: Create > Create Page
  3. Choose “Fiat Pro Editorial Template” as page layout
  4. Click Next
  5. Insert page data (a title can be “Camper”)
  6. Click “Create”;
  7. Click “Done” in the opened popup

HOW TO CONFIGURE THE VIEWER COMPONENT

In order to add Viewer Module component into the page follow these steps:

  1. Click toggle side panel on the page ribbon
  2. Select “Components” tab
  3. Choose Viewer Module component
  4. Drag and drop the component in the page.
  5. Click on the “Configure” icon in order to set the information of the component.
  6. In the dialog, configure these fields:
  • General
    • Title: the title of the section
    • Description:the description of the section
    • Lower description: the price value for the section
    • CTA description:the CTA description for the section
    • CTA label:the CTA label for the section
    • CTA linked page: the CTA linked page for the section
  • Video
    • Video preview (desktop): the path of the video preview image shown in desktop view
    • Video preview (smartphone landscape):the path of the video preview image shown in smartphone landscape view
    • Video preview (smartphone): the path of the video preview image shown in smartphone view
    • Video code: the code of the video
  • Background
    • Alt image: the message that appear if image inserted is not longer present
    • Background image (desktop):the path of the upper background image shown in desktop view
    • Background image (tablet):the path of the upper background image shown in tablet view
    • Background image (smartphone landscape):the path of the upper background image shown in smartphone landscape view
    • Background image (smartphone): the path of the upper background image shown in smartphone view

CTA

  • General
    • Single promotion label: the label for single promotion in the first box
    • Promotions label:the label for multiple promotions in the first box
  • Second tab (the fields of this tab are required)
    • CTA label:the label of the CTA
    • CTA linked page:the path of the linked page for the CTA
  • Third tab
    • CTA label: the label of the CTA
    • CTA linked page:the path of the linked page for the CTA
  • Fourth tab
    • CTA label: the label of the CTA
    • CTA linked page:the path of the linked page for the CTA
