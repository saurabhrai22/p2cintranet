HOW TO CREATE THE NEED BASED TOOL PAGE

In order to create the Need based tool Page, follow these steps:

  1. Select the language page (e.g. fiatprofessional/ fiatprofessional-it/it)
  2. Click: Create > Create Page
  3. Choose “Fiat Pro Website Need Based Tool Template” as page layout
  4. Click Next
  5. Insert page data (a title can be “needbasedtool page”)
  6. Click “Create”;
  7. Click “Done” in the opened popup

HOW TO CONFIGURE NEED BASED TOOL PAGE

In order to configure the Need based tool page follow these steps:

  1. Select the “it” page (e.g. fiatprofessional/ fiatprofessional-it/it/gamma-fiatprofessional/)
  2. Select the page whit a template “Model Equipment Template”
  3. Click “Properties”
  4. In the “NEED BASED TOOL” tab configure this field:
Configuration
  • The checkbox for show/hide need based tool:Hide from Need based tool:
Filter 1 configurations
  • Filter 1 tags:the tags for the filter 1;
Filter 2 configurations
  • Filter 2 tags:the tags for the filter 2;
Filter 3 configurations
  • Filter 3 tags:the tags for the filter 3;
Filter 4 configurations
  • Filter 4 tags:the tags for the filter 4;
Filter 5 configurations
  • Filter 5 tags:the tags for the filter 5;
Filter 6 configurations
  • Filter 6 tags:the tags for the filter 6;
Editorial
  • Title:the title of section;
  • Description:the description of section;

IMAGES

  • Alt Image:the message that appear if image inserted is not longer present;
  • Image (desktop):the path of the image shown in desktop view;
  • Image (smartphone landscape):the path of the image shown in smartphone landscape view;
  • Image (smartphone):the path of the image shown in smartphone view;

Top features

  • Icon 1the path of icon 1;
  • Title 1:the first title of section;
  • Subtitle 1:the first subtitle of section;
  • Icon 2the path of icon 2;
  • Title 2:the second title of section;
  • Subtitle 2:the second subtitle of section;

HOW TO CONFIGURE DEALER LOCATOR VIEWER IN DEALER LOCATOR PAGE

This component is provided by “Fiat Pro Website Need Based Tool Template”, configure the fields in dialog as shown below

GENERAL

  • Configuration icon: the icon of configuration
  • Start Configuration Label: the start label
  • Configuration description: the configuration description label
  • CTA Results label: the CTA result label
  • CTA Label: the CTA result label
  • Searches of your needs label: the searches of your need label
  • Rotate label: the rotate label

STEP X

Settings
  • Hide step: the checkbox for show/hide the step
  • Tags Order: the order when the tags are displayed
  • Item x: is the item of the step x *
  • Read subdata: the checkbox for show/hide the subdata. If the checkbox is checked, in “Tags order” select only a tree tag (for example “Esigenze specifiche” or “Dimensioni esterne”)
On Item x you can choose about these items:
  • Car spaces
  • Tractions
  • Cargo Area
  • Loading Cargo
  • Length
  • Fuel Types
General
  • Question: the question of the step x
  • Recap title: the recap title of the step
  • Icon : the path of the icon shown in desktop view
  • Icon (on mobile): the path of the icon shown in mobile view
