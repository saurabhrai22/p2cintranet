FIAT

  • Spends note a shift of +5% to € 4,879.09 with +9% increased traffic and slight change in the cost per click -4%. Monthly budget for April was € 21,720.00, out of which 64% has been consumed noting a -9% underspending.

Points that will help to write insights:

  1. Evaluate the reports – starting with the leads > relating it to the change in traffic > changes noted in cost per click > how it has affected the Spends and ultimately the cost per lead.
  2. Observe the %change {formula: =(current week value - last week/last week)} and difference in numbers WoW for all the tabs – brand type wise, remarketing stage and model wise – as one or more of those might have affected the changes
  3. Point number 2 will roughly give an idea over the model which has affected. To find the root cause - apply comparison with the last week to find the exact campaign.
  4. Check the Graph representation to know the date on which the changes were affected
  5. Change history will help to know the tentative reason that would have affected the changes
    1. Even Google Trends help to know the reason if not supported by Change history
    2. Search term report will give keywords that were out of the campaign focus yet getting clicks and increasing the CPC
  6. 6. Briefing of insights –
    1. Starting with the change observed in spends with relation to traffic (clicks) and cost per click. The 1st point will also support the utilization of monthly budget calculated in the ‘Over-Under Spend Calculator_(BE)’
    2. 2nd Point will have the changes in leads - effect on CPL – change in lead rate if required co-relation to traffic also can be mentioned with the performance of the lead generating parameters.
    3. 3rd Point – Model wise changes to be mentioned in terms of leads (always write about the device that recorded highest leads 1st) – if all models have had conversions; we can mention the ones that affected the most and latter comment for brand generic and non-brand. If no conversions were recorded then change in CTR and CPC affected majorly can be scripted
    4. Next Point will have the changes in permissions and the performance of the converting parameters.
    5. 5th point is for remarketing users – the point will be supported highlighting the leads if any in relation to the traffic; if no leads then to comment on the changes in CPC and CTR
    6. Last point to be jotted for the Impression Share of Brand Campaigns and the effect from the IS lost by budget.
    7. g. And at the End the Change history pointers will be jotted down

**As far as possible write the positive points first. While mentioning the change in percentage apply – or + symbols to define the trend
