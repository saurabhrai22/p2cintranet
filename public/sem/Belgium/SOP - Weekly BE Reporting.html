 The points needing to be noted while creating reports:

  1. What need to be changed for Raw data –
    1. Ad word report raw data {BE - Weekly - Search IS - 5 Brands (for all brands)}: remove top 5 rows and totals
    2. Search Impr. Share and Search Lost IS (budget) replace < 10% with 10% and > 90% with 90%
    3. Select “Keyword” column and replace ] and [ and “ with none.
    4. Format Raw Data: Del 0 Impressions (** Only Fiat has 2 keyword Data, delete the data having lesser value).
    5. Save the file in CSV format.
    6. Ensure that the metrics in the excel downloaded from Marin reflect AdWords metrics on dashboard. In case of any discrepancy then please escalate the issue with Onshore team.



Automations:

  • Open Automation: Click Run Button (close all the files open in the backend and remove filters).
  • If Automations stops due to Not Found Campaigns – Complete the mapping.
    • Mapping at the Campaign + Ad Group Key.
    • 1st Map Generic campaigns – filer GNRC.
    • Then map model type according to the models for the brand.
    • Data pending after mapping Models are Brand Generic keywords.
  • Re-Run the automation (close all the files open in the backend and remove filters).
  • Tally automation data with Google Dashboard.
  • Vlookup IS for Top keywords with the BE- Weekly IS Report.
  • Update the IS for Models, Luxemburg and Brand Type tabs - Filters for the same is as below and mostly GDS dashboard can be used for ease of work.
  • Remarketing Data – Account Totals and Remarketing Data both from Google Dashboard.
  • Group Weekly and Luxemburg Tab to display 10 weeks data.
  • **Summary Reports will be updated from the Macro Automations.
  • **Summary sequence and pattern to be followed as BE- SOP - Weekly Insights. Pointers related to Marginal or nominal changes can be skipped.

DO’s:

  • Tally the automation numbers with the Google Dashboard for all the levels.
  • Check the formulas at Weekly, Luxemburg and Remarketing Tab.
  • Check the formatting of Keywords at Top Keywords.

 Formatting:

  1. Copy paste ‘This Week row data to ‘Last Week’ row
  2. At the WoW grouping (historical) data – Enter a new row before the last Week’s row.
  3. Copy the entire last row and paste values to the new row.
  4. Delete data from the last row
  5. Enter the date range in the last row, according to which period the reports will be prepared without changing the format
  6. Group the data to represent 10 Weeks data (* last row being blank).
