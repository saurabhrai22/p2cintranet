  • The points need to be noted while creating reports:

  1. Copy-Paste previous week folder and rename it with present week date (ex. From Jan 01 – Jan 07 to Jan 08 – Jan 14)
  2. Change the dates of the reporting spreadsheets and insights document to present reporting week dates
  3. There are 2 different Automation sheets
    1. Retail - “ETE_Automation All Brands (IT)_v5”
    2. Fleet – “ETE_Automation All Brands (IT FLEET)_v2”

  1. When using Retail automation - “ETE_Automation All Brands (IT)_v5”: There are 3 run buttons in “Raw Data” sheet.
    1. Click on “Run” button to create regular weekly spreadsheets, remarketing spreadsheets with Top keywords
    2. Click on “Run Fiat Pro” button to create Fiat Professional weekly and remarketing spreadsheets
    3. Click on “Run Top Ads” button to create Top Ads for 6 regular weekly spreadsheets
  2. While using Fleet automation - “ETE_Automation All Brands (IT FLEET)_v2”:
    1. Just Click on “Run” button to create all fleet weekly spreadsheets with Top keywords and Top Ads

  • Region Report:

    This report needs to be prepared manually using “IT - Geographic Report - 9 Accounts” raw data and formulated automation sheet “Automation Region Report (IT)”.

    1. Change the Account names to “01 – Fiat”, “02 – Jeep”, “03 – Alfa Romeo”, “04 – Lancia” and “05 – Abarth”
    2. Change “Country” column to “Brand Type” and map campaigns to reflect “Brand or Generic”
    3. Add a pivot:
      1. Account to Rows
      2. Brand Type to Columns
    • Regions to Rows
    1. Add Clicks, Cost and Conversions to Values
    1. Copy and paste individual brands to “Automation Region Report”
    2. Manually format “Weekly Region Report” to make sure that we are adding data (Conversions, CR and CPL) from automation sheet to the present week of region report
  • Summary Report:
    Use the spreadsheets created through automation and copy-paste brands WoW total change data to respective brand table created in the summary report.
  • Over-Under Spend Calculator_(Day wise):
    Update the “Orange” cells to reflect spends for the present week:
    1. Change the No. of days
    2. Total budgets to reflect account budgets in AdWords
    3. Spends should reflect actual spends for the no. of days mentioned in step 1 in AdWords
