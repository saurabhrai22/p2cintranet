Customer Journey Report

  • After receiving the Keyword Mapping:
  • Add the mapping to the Mapping Sheet of the respective brand.
  • For, Brand Type and Model Type of the added list; (Key – Campaign+Adgroup) update the mapping
  • Recheck the Kwd Raw Data Tab – Not Founds should not be there
  • Verify the checker: The Difference column should be “0”
  • If there is Difference – Remedies:
    • At times the onshore team may change the format of Kind Keyword, Check with the earlier mapping. For Eg., ‘Brand Offer’ may be written as Brand+Offer; Formula being updated as ‘Brand Offer’, Brand+Offer will not be read. Change it accordingly.
    • If the above is not an issue; Use the Trim Formula and update the Kind of Keyword Column.
    • There could be a scenario where the mapping is also incorrect. For Eg., Model Keywords from Evaluation Phase mapped as Brand Pure; Check the keyword, Campaign and Ad Group and Replace it with Model Pure.
    • There also could be a chance that the type of keyword with the respective match type has not been updated in the reports, update the report table accordingly with the formulas
    • To Verify the data, Filter Inspiration phase and cross check the numbers for Clicks, Impressions and Cost with the data populated in the Inspiration Tab for the brand. Similarly, cross check the conversions

Formatting (Monthly CJ Report):

  • Copy the data (from Date Range) 1st table-current month and paste it at the date range of currently seen 2nd week.
  • Del the data for the data seen in the 1st table (inspiration tab)
  • Update the top Date Range: according to the period, for which reports will be prepared without changing the format
  • Del the data for all tabs from the current months table.
    • **2 Months rolling data.

Reporting (Monthly):

  1. Update the 1st table in inspiration tab with the 1st table data in automation-Insipiration Tab.
  2. Copy paste(values) the data of individual phase to the brand’s report {Expand the table (if Grouped)}.
Note: While saving the file, curser will be in the Date Range: cell for all tabs and displaying the Inspiration tab.

PPT Automation Sheet:

  1. Format the Automation_CJ-PPT (CJ-Fiat, CJ-Jeep, CJ-Alfa Romeo, CJ-Abarth)
  2. Copy the data (from Date Range) 1st table-current month and paste it at the date range of currently seen 2nd month.
  3. Update the top Date Range: according to the period, for which reports will be prepared without changing the format
  4. Del the data for all tabs from the current months table
  5. Copy paste the data for current month from CJ Reports and paste it according to the respective brand and Phase.
  6. Extract the Channel performance data from the weekly reports.

PPT (Monthly):

  1. Cover Slide – Update the Month (PPT is being prepared for) Date (on the day report will be sent)
  2. Copy the Pie Chart (Brand Wise) in Inspiration Tab from automation sheet and Paste (Keep Source Formatting and Embed Workbook). Delete the older pie chart and adjust the new pie chart in that area.
  3. Update the data for Device Data and Phase wise
  4. For Top Models, Evaluation:
      Evaluation Phase > Group the data > Model with highest Actions > Compare the Actions and CPA for the model of this month with the Actions and CPA of the same model last month and enter the required data with the Name and Picture {available in the CJ PPT Data sheet} of the brand in the automation file.
  5. For Top Models, Confirmation:
      Confirmation Phase > Group the data > Model with highest Leads> Compare the Leads and CPL for the model of this month with the Leads and CPL of the same model last month and enter the required data with the Name and Picture {available in the CJ PPT Data sheet} of the brand in the automation file
    6. Note: If there are no leads or conversions for the phase enter NA all cells. If there is no data for the model in the previous month to compare, then 0%.
  6. Adjust the Downward and Upward arrows for Device Table:
    1. Negative Spends, Leads and CR – Red Downward Arrow;
    2. Positive Spends, Leads and CR – Green Upward Arrow;
    3. Negative CPL - Green Downward Arrow;
    4. Positive CPL – Red Upward Arrow
  7. Select all the horizontal Arrows in one row and Format>Allign> Allign Middle
  8. Select all the Vertical Arrows in one column and Format>Allign> Allign Centre
