Common errors:

The most common reported errors are the following:

  • adobedtm script of another country inserted on the portal
  • scripts inserted more than one time in the pages
  • script disabled or commented

Checks of correctness:

  • Inspect the code of the page
  • Verify if the adobe DTM script appears only one time and after the cookie law section in the page
  • Verify if the adobedtm script is correct for market/brand/device/environment (check ?Script Adobe_configurators_All brands.xlsx? file)
  • Verify if the pagebottom script appears only one time in the page
