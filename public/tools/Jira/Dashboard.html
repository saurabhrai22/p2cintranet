3. HOW TO CREATE DASHBOARD

  1. Under << Tools >> select << Copy Dashboard >>
  2. Give it a name
  3. Click << ADD >>

4. HOW TO EDIT COMPONENT ON DASHBOARD

  1. Select the dropdown icon on the right and select << Edit >>
  2. Fill in your own filter. See
  3. Click << Save >>
  4. Click << ADD >>

5. HOW TO ADD NEW GADGETS ON YOUR DASHBOARD

  1. Select << Add Gadget >>on your dashboard
  2. Click on << Load all gadgets >>
  3. Select the gadgets you want
  4. Note: it is possible to add the same gadget multiple times
  5. Click << ADD >>

6. HOW TO CREATE FILTER

  1. Select a filter created earlier.
  2. Adjust the filter. Check if tickets is valid and tickets are found.
  3. Select << Add Gadget >>on your dashboard
  4. Click on << Save as >>
  5. Give the Filter a name
  6. Click << Submit >>
  7. Note:it is possible to have filters with the same name.
  8. Click << ADD >>
