The goal of this document is to provide the basic tips in order to create Form Eloqua in the Jeep portal.

2. HOW TO CREATE FORM ELOQUA

    Follow the steps below:

  1. Go to the properties of the language node (ex. jeep/jeep-uk/en);
  2. Select tab “sites configuration”
  3. Under the field “Eloqua Form Configuration” insert the Landing Page ID
  4. Launch this servlet in order to import the Eloqua form:
    1. http://author-fca-italy-testing.adobecqms.net/content/jeep/jeep-uk/en/_jcr_content.eloquasync.json 
      Note: Change market (ex.jeep-uk) and language (ex. en) in the url for each market and language
  5. Under the properties of the language node, select the tab “Form Standard Reference”
    1. For each form insert the path to the model page that can be selected in the form page
    2. Insert the path to the form page
  6. Choose “Jeep Form Eloqua MARKET template” as page layout (choose the template for the relative market)
  7. Click Next
  8. Insert page data (a title can be “Test drive”)
  9. Click Create
  10. Click Doneat the opened popup;
  11. Insert the “From Eloqua” (standard or business)
  12. Select the type of the form in the drop down menu in the dialog of the component

3. HOW TO CONFIGURE THE OUR SHOWCASES COMPONENT

Follow the steps below in order to create the “model lookup page”:

    1. Go to the language node of the CMS editor ( jeep/jeep-uk/en);
    2. Click: Create > Create Page
    3. Choose “Jeep Redirect Template” as page layout
    4. Click Next
    5. Insert page data (a title can be “model lookup page”)
    6. Click “Create”;
    7. Click “Done” in the opened popup
    8. Click “Edit”;
    9. In the “Basic”;” tab check the “Hide in Navigation”; checkbox.
    10. Click “Done”

      To create the “model page”, follow the steps below:

    1. Go to the model lookup page of the CMS editor ( jeep/jeep-uk/en/ model-lookup-page);
    2. Click: Create > Create Page
    3. Choose “Jeep Website Model Lookup Template” as page layout
    4. Click Next
    5. Insert page data (a title can be “Renegade”)
    6. Click “Create”;
    7. Click “Done” in the opened popup
    8. Select the page created before, Click “View Properties”;
    9. Click “Edit”;button
    10. Under the tab “Analytics”, insert the Model description Adobe and the Version description Adobe
    11. Under the tab “Model Info”, select the model and the version from the drop down menu; the other fields will be autocompleted.
    12. Under the tab “Model Data”, insert the description title, the images and the link to the thank you page CTA (field “thank you model CTA”)
    - Display title;
    - Brochure;
    - Thank you model page CTA: here you have to choose the model page under the Jeep Vehicles node as in the picture below;

    Note: The field “Image Min 1600” is for the image inside the “Thank you page”

    4. HOW TO INSERT FORM LABELS

      In order to modify the Eloqua form’s label follow the steps below:

    1. Go to the language node (eg. content\jeep\jeep-uk\en)
    2. Select View properties
    3. Select the tab “JEloqua form label” as page layout
    4. Under"Eloqua form label” tab there are 4 boxes: Message, Dealer, Form, Thankyou
    5. In order to modified or added the labes click “Edit”

    In the first box “Message” insert: 
    - var message

    In the second box “Dealer” insert: 
    - var formLabel 
    - var select 
    - var tabName

    In the third box “Form” insert: 
    - var dealerMessage

    In the fourth box “Form” insert:
    - var typ_message

      All the details about the labels are described in the ppt below:

    5. HOW TO INSERT ANALYTICS PARAMETERS

      Follow the steps below in order to change the analytics parameters for the STANDARD forms:

    1. Go to the language node of the CMS editor ( jeep/jeep-it/it);
    2. Click View properties
    3. Select the tab “analytics”
    4. In order to modified the parameters click "Edit”
    5. In the bottom of the page, in the last three fields insert:
    - business Area :"Sales"
    - Campaign ID : specific for each market
    - Offer ID: specific for each market
    1. Click “Done”;button

      Follow the steps below in order to change the analytics parameters for the BUSINESS forms:

    1. Go to the business node of the CMS editor ( jeep/jeep-it/it/business);
    2. Click View properties
    3. Select the tab “analytics”
    4. In order to modified the parameters click "Edit”
    5. In the bottom of the page, in the last three fields insert:
    - business Area :"Sales"
    - Campaign ID : specific for each market
    - Offer ID: specific for each market
    1. Click “Done”;button
