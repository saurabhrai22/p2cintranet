Uploading DEM in Eloqua

If everything is OK follow the below steps of uploading a DEM in Eloqua.

  1. Download the assets and create .zip file of the HTML and the Images from the downloaded assets which are saved locally on your computer.
  2. Now access Eloqua and Select Emails submenu from Eloqua dashboard.(Please ensure you already have necessary access to Eloqua)

  3. Click Upload an Email from the newly opened window and further click on Upload Email (see screenshot).



  4. In the next step click the cloud icon and Browse the .zip html asset file, once its selected click Open.



  5. The next screen shows the assets from the zip file that need to be saved. Save them in the correct folder location by navigating to it.
  6. Inorder to choose the correct path click the folder icon in top right of the screen.



  7. In the next screen select the folder for storing the images. The folder structure will be as below

    8. All -TEST-Fiat Campaign-Country-Brand-Year-Month

    Click on Choose button after selecting the suitable folder.

  8. Now notice the folder is available in the top right dropdown and click Next Step to further initiate asset upload.
  9. In this step mark the “Upload New Image” radio button for every file in that window. Click Next Step



  10. In the next screen click on Track All button and further click Next Step.


  11. This step is the final step of uploading DEM on Eloqua. In this step we need to perform following tasks
    1. Put Campaign name from Req. sheet in the Name field (Remove the extra spaces if any in between the name)
    2. Select the folder to save the HTML file. Below is the folder path to save the HTML file.

      3. All – EMEA – Country – Brand – Year – Month

    3. Select the Blank Email Group from the dropdown. Click on “Finish” button.


  12. After clicking Finish, you will be directed to the email editor page where you can update header and footer by following below steps.
    1. Enter theSubject line as per Requirement sheet
    2. If the header, given put the below link for it to open the DEM in the browser view

      3. <a href=”http://app.promo.fiat.com/e/es?s=<eloqua type='emailfield' syntax='siteid'/>&e=<eloqua type='emailfield' syntax='elqemailsaveguid'/>&elq=<eloqua type='emailfield' syntax='recipientid'/>”>

    3. Check if footer is given or not. Please put the footer text from the below documents

      4. DEM :  Click Here

      Newsletter: Click Here

      Put the unsubscribe link as per EMAIL_SETTING file



  1. Put the settings from Actions tab as per the EMAIL_SETTING Excel file. You can filter out the settings as per country and brand. Here you can put the dynamic header as per settings file (if the header is not given). Then put the preview text (if given in Req. sheet), Sender Name, Sender Email, Reply to Name, Reply to mail, and Bounce back from settings file. After filling the above field click Done


  1. Now put the links (if given in Req. sheet) to the respective buttons or hyperlinks in the Emailer.

While putting the links keep in mind the below things

  1. ?elqTrack=true [if ? present in the link then use &elqTrack=true at the end]
  2. &elqTrack=true [if ? not present in the link then use ?elqTrack=true at the end]
  3. If link is redirecting to landing page or it is an unsubscribe link then ignore step i and ii.
  4. If the link has http://dpromo..... Then refer the below points

    For FIAT ITALY: dpromo => promo

    For ALFAROMEO ITALY: dpromo => promozioni

    Rest All BRANDS AND MARKETS: dpromo => epromo

  5. If the has https://dpromo..... Then keep it as it is.
  6. Make sure every link has “target=_blank”

  2. Save the asset after performing the above steps
