Special Requirements

  1. Dynamic Fields

    2. Some times there is requirement to put the dynamic field for name, surname etc

    For this purpose you need to select field merge or Dynamic field from the editor. In that, you need to drag the specific field wherever required.

      Some commonly used dynamic fields are

    1. First name – Search for “First Name used for Email only” in the field merge
    2. Last name – Search for “Last Name” in the field merge
    3. For gender you need to use Dynamic content and drag the necessary field


  2. DEM Resend

    3. Some times as per given in the Req. sheet, you need to create the Resend copy of DEM. Generally it is for the users who did not receive/open the first DEM. For that purpose you need to create the clone(Copy) of first/main DEM and change the subject if new subject line for resend is given in the Req. sheet.

  3. FCA HERITAGE AND EGAZZIATA FIAT
    1. For FCA Multibrand DEMs, Sender name will be always FCA HERITAGE. (Confirm with Reporter)
    2. For Fiat – France:- Please check for eGazziata Fiat to be put in Sender name. (Confirm with Reporter)
