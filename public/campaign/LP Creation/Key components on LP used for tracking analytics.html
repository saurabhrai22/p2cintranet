Key Components on LP used for Tracking/Analytics

  1. Meta Tags:
    Meta tags contain some very important information about the LP.
  2. adobedtm script:
    These is an external script which is specific to country and brand (Even sometimes specific to model as well). It is used to track the page and for analytics purpose. For e.g. script link will be different for fiat Germany and abarth Germany.
  3. Cloudfront script: Cloudfront script also called as opentag script, is used for analytics as well. These also changes with country, brand and model.
  4. Datalayer: It is also used for analytics purpose. It is implemented through coding on LP, Dealer Page and Thank You page as well.
  5. Dynamic MK js: This script auto fetches MK meta values

All above tags and scripts have some syntax and format of implementation.
