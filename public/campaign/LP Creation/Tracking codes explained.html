Tracking Codes Explained

  1. adobedtm script : adobe digitaldata function collects some vital analysis data. It is basically JSON object which submits/pushes data to the dashboard/server via a script. Some of the values are to be hardcoded and some are collected through META tag values. JS script link is selected based on brand and country. Following link has all scripts till date, however new script links keep adding up as the countries increase. However these links are different for staging and production environment, so it has to be properly selected.
  2. maxus opentag: These works on similar grounds as adobe. But it captures some different parameters for Analysis and tracking. It also uses a JS to push/submit data from page to server/dashboard. It is implemented as a JSON function and a JS link on page. JS links are specific to Brand and country.

                        <script src="//d3c3cq33003psk.cloudfront.net/opentag-97303-1177618.js" async="" defer=""></script>
