Analyzing (Reading) Requirement Sheet

Requirement sheet is very important as it determines the implementations and development details for Landing Page. It covers almost all details pertaining to the Landing Page development. So it is more important that the requirements are properly mentioned and understood in requirement sheet.


Following is a sample link to a Requirement Sheet.

Sr No Parameter Description Possible Value Implementation
1 Type of landing page Determines the template to be used for developing New LP. This template is chosen based on some parameters such as country, brand, Creativity and state of LP(Production or Certification) so that minimum changes are to be made. Link of previously developed LP LP is copied and cloned for developing new LP
2 Campaign Name Determines the type of campaign Promo or Blank(no promo) <meta name="CAMPAIGN" content="5522">
3 Brand Determines Brand for which LP is created. Fiat, Abarth, Jeep, AlfaRomeo, Lancia. <meta name="Brand" content="FIAT">
4 Brand Code Determines the Brand Code for respective Brand. These values remain same for a specific brand Fiat – 00 Abarth – 066Jeep <meta name="BRANDCODE" content="00">
5 Car Model Determines the Model for the Brand. A brand has many models. New models keep on adding .There is no specific set of models defined as such. 500 Qubo Panda <meta name="MODELDESCRIPTION" content="500">
6 Car Model Commercial Code Determines the commercial code for the model. It is used for identifying the model with these unique ID which is defined by FCA (Fiat Chrysler Automobiles). e.g.: 500X – 334 Mito – 145 <meta name="MODELCOMMERCIALCODE" content="334">
7 Car Model Description Determines the Model for the Brand. It can be name given to a model for a specific promotion or collection. e.g.:
500Cabrio
500 Vintage
 <meta name="MODELDESCRIPTION" content="500C">
