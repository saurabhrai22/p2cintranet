Master Key Implementation on LP

Master Key concept determines the Navigation of visitor coming on to Landing Page. It helps to track from which source visitor land to the landing page. These can be from some banner, video, promos, retargeting, organic search, paid search, Newsletter, retargeting and many more. This is achieved through META tags and javascript which automatically calls these values.

Following are the Steps to understand and implement a Master Key :

Master Key PDF have 4 Levels defined, LEVEl1, LEVEl2, LEVEl3 and LEVEl4. Level2 determines/checks source and based on its value other levels are reset.

Note: Due to DMK implementation i.e. dynamic master key script we do not manually hardcode the MK values and its updated dynamically. Also make a note that in CO Level1 value=landing page, level2 value=brand:market:contact, level3 value=CTA, level4 value =band, these values get captured always.
