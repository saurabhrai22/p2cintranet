Campaign Creation

  1. Go the home page, click on the 2nd tab of Orchestration and select Campaign.


  2. Double click on “create a multi-step campaign”.


  3. Drag the “Segment members” step on to the canvas.


  4. Double click on the segment members.


  5. In the choose a segment section, write the name of the segment created previously. It will automatically pop up in the drop down. Copy the same name in the “Step name”.

    Counts will appear once you finish with the above steps.



  6. Next step is to drag the shared list step to the canvas.


  7. Double click on the “Add to Shared list”, select new and give the same segment name and save in the designated folder.


  8. The next step after creating shared list is to give tracking id to every shared list. For that you need to go to home page> tools (in the 4th tab of Audience)>shared list.


  9. Search for the shared list created.


  10. Double click on the shared list (For eg: Customers Segment Test), click on the arrow on the left side and select settings.


  11. The last step is to put the tracking code in the description, click and done and then save.


  12. Every segment have different tracking ids. Kindly find the list below.
