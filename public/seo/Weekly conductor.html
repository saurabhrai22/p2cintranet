1. General Instruction

  • Keyword Ranking report (Conductor Report): Use Searchlight conductor tool
  • The brands to be reported are Fiat, Jeep, Alfa Romeo, Lancia, Abarth and Fiat Pro and locales are Italy, France, Germany, Belgium, Netherland, Denmark, Poland, Austria & Spain.
  • Reporting to be done separately for Desktop and Mobile.

  • Nomenclature for this report used is as follows:

    Conductor_KW_Drop_Report (Local Name) _WK_Start day of week_Oct'17 – Last day of week_Oct'17

2. Reporting Data for Keyword Ranking Report

Follow the below steps for preparing a weekly report for keyword rankings:

  • Login to Searchlight Conductor tool
  • Select appropriate account to be reported for the week – France or Italy or Germany etc.
  • Go to Measure Visibility -> Visibility Performance to land on the below dashboard
  • Always use Standard Rank
  • Use appropriate channels – Desktop or Mobile from above and below images resp.
  • Consider appropriate date as per the week as in below images
  • Extract data (download) from below 4 drop reports:
    • Drop off 1st pages
    • No longer ranking for
    • Dropped out of Top 3
    • Drop from Position 1
  • Place the appropriate data in the excel reporting sheet
  • Sort (large to small) using Monthly searches. Place all occurring NA’s at the last
  • Ensure proper formatting of the data in each tab of the excel (Font Calibiri – Size 11)
  • Follow these steps for all the markets to prepare weekly reports
  • For BE market, we must prepare two files for two different languages (French and Dutch)
  • For IT Market, while working on mobile channel report select only below mentioned city.

Note: Always hide the brand tab if data is not available for that week & mention in the email while sending to the on shore team
