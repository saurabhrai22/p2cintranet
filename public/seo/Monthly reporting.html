Monthly Reporting

1. General Instruction

  • Reporting of Monthly Performance Reports and Technical work books
  • Monthly Performance Reports to be delivered in 1st week of every month
  • Technical work books to be delivered in 2 days when requested by the onshore team
  • Monthly Performance Reports: Adobe Analytics tool to be used for reporting the data
  • The brands to be reported on are Fiat, Jeep, Alfa Romeo, Lancia, Abarth and Fiat Pro and locales are Italy, France, Germany, Belgium, Netherland, Denmark, Luxemburg, Poland, Austria & Spain.
  • Reporting to be done separately for Desktop and Mobile channels
  • The standard templates for reports and technical workbook document are to be adhered to
  • Nomenclature for the reports and documents is as follows:
    • Monthly Reports
    • For Monthly reports the format would be
    • FCA-SEO-Country Monthly Performance Analytics_Month_YYYY
    • Example: FCA-SEO-Italy Monthly Performance Analytics_Jan_2016
  • The report and technical document is to be shared with Mumbai SEO Lead before it goes to the market SEO Lead for review.
  • The turnaround time for all the Monthly reports is 5 working days from the start of the month and for technical workbook it is 2 working days from the day of request.
  • When referring to any considerable account change that may have impacted a KPI on the Monthly reports, it is mandatory to include comments stating reasons for the fluctuations.
  • Once the Monthly report is ready, verify it against Adobe analytics and Google webmasters for accuracy

From Adobe Analytics

3.1 Extracting the data

For extracting the data from Adobe Analytics, follow below steps:

Step 1: Login

Open URL (https://sc3.omniture.com/login/) >> Enter Company, Username and Password >> Click Sign In button

Step 2: Select Dashboard

Follow below steps:

  1. Click on Menu Icon
  2. Click on Components >> Manage Dashboards from the list
  3. Search for the Dashboard in the “Shared Dashboards” list >> Select it

Note: Names of dashboard currently in use:

  1. SEO KPI Dashboard - Desktop - Mar-17
  2. SEO KPI Dashboard - Mobile - Mar-17
  3. SEO KPI Dashboard - Desktop - Mar-17 –BE
  4. SEO KPI Dashboard - Mobile - Mar-17 –BE
  5. SEO KPI Dashboard - Desktop - Mar-17 –LUX
  6. SEO KPI Dashboard - Mobile - Mar-17 –LUX
  7. Global Analytics Report – New -0rganic Segments

For the mentioned dashboards, use below segments:

  1. SEO-Organic Traffic (new)
  2. Corporate Website pages-Desktop (Visit Level)
  3. Corporate Website pages-Mobile (Visit Level)
  4. Belgium HQ - visit level
  5. Luxembourg HQ - Visit level
  6. Corporate website pages (Test)
Step 3: Extract Overall Total Visits, Bounce Rate and Average Time Spent on Website

Follow below steps:

  1. Select appropriate suite (Standard - EMEA Country Brand Tier 1) and Date Range
  2. Copy Visits, Bounce Rate and Dwell Time from report let Key Metrics Report – Overall Total Visits to excel report.

Note: Dwell Time is Average Time Spent on Website

Step 4: Extract Organic Visits, Unique Visitors, Bounce Rate, Page Views and Average Time Spend on Website

Follow below steps:

  1. Select appropriate suite (Standard - EMEA Country Brand Tier 1) and Date Range
  2. Copy Organic Visits, Unique Visits, Bounce Rate, Average Time Spent and Page Views from reportlet Key Metrics Report – Organic Total Visits to excel report.

Note: Dwell Time is Average Time Spent on Website

Step 5: Extract Search Engine Data (Google & Bing)

Follow below steps to get Google Data:

  1. Select appropriate suite (Standard - EMEA Country Brand Tier 1) and Date Range
  2. 2. Copy Total Visits from reportlet Search Engines – Natural Report Google Only to excel report.

Follow below steps to get Bing Data:

  1. Select appropriate suite (Standard - EMEA Country Brand Tier 1) and Date Range
  2. Copy Total Visits from reportlet Search Engines – Natural Report Google Only to excel report.
Step 6: Extract Top Organic Landing Pages

Follow below steps

  1. Select appropriate suite (Standard - EMEA Country Brand Tier 1) and Date Range
  2. 2. Copy Page URLs, Visits, Bounce rate & Average Time Spent from reportlet Page URL Report – Top Organic Landing Pages to excel report.

Note: Dwell Time is Average Time Spent on Website

Points to keep in mind while working on this report:

  • If two URLs are similar then add all required data values. (For e.g https://www.alfaromeo.de/giulia & https://www.alfaromeo.de/giulia#_ )
  • If site turns to responsive and getting equal number of visits for both domain data then add all required data values. ( https://www.alfaromeo.de/ andhttp://www.alfaromeo.de/ )
Step 7: Extract Top Referring Domains

Follow below steps

  1. Select appropriate suite (Standard - EMEA Country Brand Tier 1) and Date Range
  2. Copy Original Referring Domains, Visits, and Average Time Spent from report let Original Referring Domains Report – Referral Domains to excel report.

Note: Dwell Time is Average Time Spent on Website

Step 8: Extract Traffic Sources

Follow below steps

  1. Select appropriate suite (Standard - EMEA Country Brand Tier 1) and Date Range
  2. Copy Other Natural Refferers, Typed/Bookmarked and Paid Search visits from report let Channel (Last) Report – Traffic Sources to excel report.
  3. Typed/Bookmarked = Direct Traffic
  4. Other Natural Referrals = Referring Domains
Step 9: Extract Events Organic

Follow below steps

  1. Select appropriate suite (Standard - EMEA Country Brand Tier 1) and Date Range
  2. Scroll down to reportlet Total Organic Event for Leads >> Right Click on report and open in New Tab

See Screenshot of Total Organic Event for Leads after right clicking on it.

  1. Check Date Range and Report suite for confirmation
  2. Copy Visits, Visitors, Lead Start, Lead Completed and Lead Submit from reportlet Total Organic Event for Leads to excel report.

Note: Follow same steps to extract all other leads and permissions. Only reportlet name will change as per the lead or permission.

Step 10: Extract Page Conversions for Different Events

Follow below steps

  1. Select appropriate suite (Standard - EMEA Country Brand Tier 1) and Date Range
  2. Scroll down to reportlet Page Conversions for Leads >> Right Click on report and open in New Tab

See Screenshot of Page Conversions for Leads after right clicking on it.

  1. Once the reportlet is opened in new tab, follow below steps:
    1. In the above snapshot, page column contains the names of the pages with the respective metrics - visits, unique visits etc.
    2. We report on the URL basis for the monthly Adobe analytics report
    3. For that, right click on the page name to open it in a new tab
    4. For desktop reports, consider the opened URLs starting with “www” and copy respective URLs with the corresponding metrics in the monthly report
    5. For Mobile reports, consider the opened URLs starting with “m.” and copy respective URLs with the corresponding metrics in the monthly report
    6. Repeat for 10 such entries to be filled in the monthly Adobe performance report
  2. Follow steps 2 & 3 for all other remaining page conversions for events like quote end, test drive, contact request, brochure request etc. Separate reportlets have been provided on the dashboard with the corresponding names

Note: Follow same steps to extract all other leads and permissions. Only reportlet name will change as per the lead or permission.
