Reporting Data for SEO Monthly Performance Analytics

Through this section we will understand the most important metrics for the process through Adobe analytics reporting tool 

Metrics

  • Total Visits
  • Organic Visits
  • Visits from Referring Domains
  • Direct Access
  • Paid Search (PPC)
  • Page views
  • Monthly unique visitors
  • Average Time on Site (Overall)
  • Average Time on Site from Organic visits
  • Overall Bounce Rate
  • Bounce Rate from Organic visits
  • Total and Organic Events
  • Traffic From Google
  • Traffic From Bing
  • Top landing Pages & respective traffic value
  • Top Referring Domains & respective traffic values
  • Page Conversions for Different Events

From Adobe Analytics

Extracting the data

For extracting the data from Adobe Analytics, follow below steps:

Step 1: Login

Open URL (https://sc3.omniture.com/login/) >> Enter Company, Username and Password >> Click Sign In button

 

Step 2: Select Dashboard


Follow below steps:

  1. Click on Menu Icon
  2. Click on Components >> Manage Dashboards from the list
  3. Search for the Dashboard in the “Shared Dashboards” list >> Select it

Note: Dashboard Names

  1. Adobe SEO-R-Desktop-FV2-Final-21-Mar-16-Mumbai
  2. Adobe SEO-R-Mobile-FV2-Final-21-Mar-16-Mumbai

 

Step 3: Extract Overall Total Visits, Bounce Rate and Average Time Spent on Website


Follow below steps:

  1. Select appropriate suite (Standard - EMEA Country Brand Tier 1) and Date Range
  2. Copy Visits, Bounce Rate and Dwell Time from reportlet Key Metrics Report – Overall Total Visits to excel report.

Note: Dwell Time is Average Time Spent on Website

 

Step 4: Extract Organic Visits, Unique Visitors, Bounce Rate, Page Views and Average Time Spend on Website


Follow below steps:

  1. Select appropriate suite (Standard - EMEA Country Brand Tier 1) and Date Range.
  2. Copy Organic Visits, Unique Visits, Bounce Rate, Average Time Spent and Page Views from reportlet Key Metrics Report – Organic Total Visits to excel report.

Note: Dwell Time is Average Time Spent on Website

 

Step 5: Extract Search Engine Data (Google & Bing)


Follow below steps to get Google Data:

  1. Select appropriate suite (Standard - EMEA Country Brand Tier 1) and Date Range.
  2. Copy Total Visits from reportlet Engines – Natural Report Google Only to excel report.

Note: Dwell Time is Average Time Spent on Website

Follow below steps to get Bing Data:

  1. Select appropriate suite (Standard - EMEA Country Brand Tier 1) and Date Range.
  2. 2. Copy Total Visits from reportlet Search Engines – Natural Report Google Only to excel report.

Note: Dwell Time is Average Time Spent on Website

 

Step 6: Extract Top Organic Landing Pages


Follow below steps:

  1. Select appropriate suite (Standard - EMEA Country Brand Tier 1) and Date Range.
  2. Copy Page URLs, Visits, and Average Time Spent from reportlet Page URL Report – Top Organic Landing Pages to excel report.

Note: Dwell Time is Average Time Spent on Website

Bounce rate will be calculated as per the calculated metrics.

 

Step 7: Extract Top Referring Pages


Follow below steps:

  1. Select appropriate suite (Standard - EMEA Country Brand Tier 1) and Date Range.
  2. Copy Original Referring Domains, Visits, and Average Time Spent from reportlet Original Referring Domains Report – Referral Domains to excel report.

Note: Dwell Time is Average Time Spent on Website

 

Step 8: Extract Traffic Sources


Follow below steps:

  1. Select appropriate suite (Standard - EMEA Country Brand Tier 1) and Date Range.
  2. Copy Other Natural Refferers, Typed/Bookmarked and Paid Search visits from reportlet (Last) Report – Traffic Sources to excel report.

 

Step 9: Extract Events Organic


Follow below steps:

    1. Select appropriate suite (Standard - EMEA Country Brand Tier 1) and Date Range.
    2. Scroll down to reportlet Total Organic Event for Leads >> Right Click on report and open in New Tab


See Screenshot of Total Organic Event for Leads after right clicking on it.

  1. Check Date Range and Report suite for confirmation
  2. Copy Visits, Visitors, Lead Start, Lead Completed and Lead Submit from reportlet Total Organic Event for Leads to excel report.

Note: Follow same steps to extract all other leads and permissions. Only reportlet name will change as per the lead or permission.

 

Step 10: Extract Page Conversions for Different Events


Follow below steps:

    1. Select appropriate suite (Standard - EMEA Country Brand Tier 1) and Date Range.
    2. Scroll down to reportlet Page Conversions for Leads>> Right Click on report and open in New Tab


See Screenshot of Page Conversions for Leads after right clicking on it.


  1. Once the reportlet is opened in new tab, follow below steps:
    1. In the above snapshot, page column contains the names of the pages with the respective metrics - visits, unique visits etc.
    2. We report on the URL basis for the monthly Adobe analytics report
    3. For that, right click on the page name to open it in a new tab
    4. For desktop reports, consider the opened URLs starting with “www” and copy respective URLs with the corresponding metrics in the monthly report
    5. For Mobile reports, consider the opened URLs starting with “m.” and copy respective URLs with the corresponding metrics in the monthly report
    6. Repeat for 10 such entries to be filled in the monthly Adobe performance report
  2. 4. Follow steps 2 & 3 for all other remaining page conversions for events like quote end, test drive, contact request, brochure request etc. Separate reportlets have been provided on the dashboard with the corresponding names.

Note: Follow same steps to extract all other leads and permissions. Only reportlet name will change as per the lead or permission.
